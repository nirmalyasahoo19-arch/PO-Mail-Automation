# PO Mail Automation (Prototype)

This project is a prototype system to automatically send
Released Purchase Order (PO) emails to vendors using Outlook.

## Features
- Upload PO PDF
- Enter PO Number
- Enter Vendor Email
- Automatic email with attachment via Outlook

## Tech Stack
- HTML (Frontend)
- Python Flask (Backend)
- Outlook SMTP (Email)

## How to Run (Local)
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Set environment variables:
   OUTLOOK_EMAIL
   OUTLOOK_APP_PASSWORD
4. Run:
   python app.py
5. Open browser:
   http://127.0.0.1:5000
